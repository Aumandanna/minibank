# MiniBank Fullstack Web Application

A full-stack banking simulation web application.

## Tech Stack
- Frontend: React (Vite)
- Backend: Spring Boot
- Database: PostgreSQL
- Deployment:
  - Frontend: Netlify
  - Backend: Railway

## Features
- User authentication
- Transfer money
- Deposit / Withdraw
- Bill payment
- REST API integration
